**User Story:** As an artist studying digital media, I want to create a digital, multimedia book so that I can share a combination of art, films, radio plays, and audio recordings in a cohesive narrative format.

**Description:** Writing a narrative, whether academic or creative, that includes multimedia elements, and cohesively combining text with images and audiovisual content can be challenging. AVAnnotate allows for the creation of a cohesive, “digital scrapbook” which combines these multimedia elements into one site. Relevant fields include Visual Arts, Digital Humanities, Creative Writing, and Journalism.

**Ingredients**
- A narrative (academic or creative)
- Multimedia elements that contribute to or enhance the narrative (images, audiovisual elements)
- Annotations for all audiovisual elements
- Tags (optional)

**Method**
1. Write your narrative - this can be done prior to beginning the “digital scrapbook,” or the text of the narrative can evolve over time as the scrapbook is built up
2. Upload audiovisual works to AVAnnotate and add any contextual information as **annotations**, with associated **timestamps**, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
3. Create and add **tags** to annotations, if desired, following Quick Start guide
4. Create interconnected, hyperlinked pages which follow the narrative and showcase images and audiovisual elements that are part of or contribute to the narrative

**Example:** [A Digital Florilegium](https://lgsump.github.io/digital-florilegium/) by Luke Sumpter

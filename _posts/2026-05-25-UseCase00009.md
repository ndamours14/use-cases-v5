**User Story:** As a podcast producer, I want to use images to annotate my podcast so that I can provide visual supplements to my audio content.

**Description:** Annotations do not have to be text-only. Images such as photographs, illustrations, and diagrams can be used as annotations to provide additional visual context to an audio file. Relevant to any fields producing podcasts and audio content.

**Ingredients**
- 1 or more audio files
- Images (photographs, illustrations, diagrams, etc.)
- Text annotations (optional)
- Tags (optional)

**Method**
1. Collect images that provide additional visual information to complement audio, and identify **timestamps** for each image
2. Upload audio file(s) to AVAnnotate, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
3. Add **annotations** to audio file, following Quick Start guide, but inserting **images** instead of text annotations
4. Add **text annotations** in addition to image annotations, if desired, following Quick Start guide
5. Create and add **tags** to annotations, if desired, following Quick Start guide

**Example:** *This is a hypothetical use case which is not currently possible to create using AVAnnotate, though it may be possible in the future or with a different tool*

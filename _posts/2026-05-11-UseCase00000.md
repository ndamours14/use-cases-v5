**User Story:** As a new AVAnnotate user, I want to learn and understand the features of AVAnnotate so that I can build my own AVAnnotate projects.

**Description:** This Documentation Project is an excellent introduction to AVAnnotate’s features, explaining how and why AVAnnotate can be used. It demonstrates both how projects are constructed (Documentation) and how projects can be effectively presented to an audience (Narrative).

**Ingredients**
- The Monster Mash - Documentation Project
- 1 or more audio or audiovisual recordings
- Annotations
- Tags
 
**Method**
1. Read the Documentation and Narrative sections on each page of the Documentation Project in order to learn about AVAnnotate’s features
2. Using your own audio or audiovisual records, annotations, and tags, you can create your own AVAnnotate project using the information gleaned from the Documentation Project

**Example**
[The Monster Mash - Documentation Project](https://avannotate.github.io/mm/) by AVAnnotate Team

**User Story:** As a dancer, I want to add visual annotations to a recorded performance so that I can include musical and dance notation with my performance.

**Description:** Musical and dance notation provide a visual blueprint to allow music and dance to be recreated. Including musical and/or dance notation with a video performance provides additional context to the performance and allows the performance to be recreated by viewers. Relevant fields include Music Performance, Dance Performance, and Music and Sound Studies.

**Ingredients**
- 1 or more videos of music or dance performances
- Musical and/or dance notation for the performance
- Text annotations (optional)
- Tags (optional)

**Method**
1. Collect or create musical and/or dance notation for the recorded performance(s), and identify **timestamps** for each notation (by line or by page)
2. Upload video file(s) to AVAnnotate, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
3. Add **annotations** to video file, following Quick Start guide, but inserting images or PDFs of musical or dance **notations** instead of text annotations
4. Add **text annotations** in addition to images or PDFs, if desired, following Quick Start guide
5. Create and add **tags** to annotations, if desired, following Quick Start guide

**Example:** *This is a hypothetical use case which is not currently possible to create using AVAnnotate, though it may be possible in the future or with a different tool*

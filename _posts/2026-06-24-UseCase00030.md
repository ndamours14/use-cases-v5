**User Story:** As a researcher working with audiovisual content, I want to create an annotated bibliography of my audiovisual resources so that I can summarize and reflect upon these resources.

**Description:** An annotated bibliography is typically a list of resources such as articles, chapters, or books, accompanied by brief annotations which summarize, reflect upon, and may quote from these resources. Like articles and books, audiovisual resources such as videos and podcasts can also be included in an annotated bibliography. These audiovisual resources can be uploaded to AVAnnotate and annotations can be added to specific timestamps of the videos, creating an interactive, multi-media annotated bibliography. Relevant to any fields utilizing audiovisual resources.

**Ingredients**
- List of audiovisual resources (videos, podcast, radio broadcasts, etc.)
- Annotations for each resource (summary, quotes, reflections)
- Tags (optional)

**Method**
1. Upload all audiovisual resources to AVAnnotate; content from sources such as YouTube, the Internet Archive, Aviary, and more can all be added to AVAnnotate
2. Create annotations for each resource (including summaries, quotes, reflections, etc.), with **timestamps** for all notes
  - The “Critical Annotated Bibliography” section of [this lesson plan](https://jentery.github.io/SWDemo/output_documents/Camlot_2021_ENGL_604B_2_-_(Literary)_Listening_as_Cultural_Technique_SYLLABUS_Fall_2022.html) includes information which may be helpful in creating your annotations
3. Add **annotations** to resources, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
4. Create and add **tags** to annotations, if desired, following Quick Start guide
5. Create a home page, if desired, to introduce the annotated bibliography and research associated with it

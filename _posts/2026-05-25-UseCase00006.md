**User Story:** As a scholar of literary recordings, I want to create an index of significant moments in a literary recording so that I can allow users to easily navigate the recording.

**Description:** Annotations can be added to audiovisual recordings that mark significant points in the recording, such as indicating who is speaking, what is being discussed, when segments of a  recording begin and end, etc. This serves as an index to allow users to easily navigate the recording. Relevant fields include English, Social Studies, Film Studies, and Music and Sound Studies.

**Ingredients**
- 1 or more audiovisual files
- Annotations
- Tags (optional)

**Method**
1. Listen to the recording from start to finish, noting significant moments worth indexing, along with associated **timestamps** for each significant moment
  - Tools like AssemblyAI can be used to generate an index or a transcript of an audio recording - if using an AI-generated index or transcript, it should be verified and edited by a human for accuracy
2. Upload audiovisual file(s) to AVAnnotate and add indexed moments as **annotations**, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
3. Create and add **tags** to annotations, if desired, following Quick Start guide

**Example:** [Index of VoicePrint: Canadian English & Literature](https://ndamours14.github.io/VoicePrintAnnotations/events/voiceprint-canadian-english-and-literature-ne/) by Natasha D’Amours

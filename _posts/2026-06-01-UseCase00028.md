**User Story:** As a student studying unconventional books, I want to analyze and review these books in a multimedia format so that I can capture the complexity of texts that challenge conventional reading approaches.

**Description:** Book reviews, particularly for unconventional books that challenge conventional reading approaches, can be created in unconventional ways. Reading/performing text from a book, uploading this performance to AVAnnotate, and reviewing the book and/or the performance of the book using annotations is one method that can be used. Relevant fields include English, Literature, Poetry, and Writing.

**Ingredients**
- Book for review
- Recording(s) of reading(s) from this book
- Annotations (reviewing the book and/or the performance)
- Tags (optional)

**Method**
1. Record yourself reading/performing a section or sections of the book for review
2. Make note of observations and analyses, reviewing the book in question and/or the performance of the book, with **timestamps** for all notes
  - [This lesson plan](https://jentery.github.io/SWDemo/output_documents/509.02_Deformance_Assignment.html) includes further information that may be helpful in constructing your book review
3. Upload recording(s) to AVAnnotate and add **annotations**, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
4. Create and add **tags** to annotations, if desired, following Quick Start guide
5. Create a home page, if desired, to introduce the book review; this may also include a written book review which supports or expands upon your annotations

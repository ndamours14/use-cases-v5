**User Story:** As a scholar studying the differences between audio and written poetry, I want to compare transcriptions and written poems so that I can understand the relationship between how a poem is written and how it is read.

**Description:** When a poem is read aloud, it can change and evolve from its written form. This can be due to the reader making intentional (or unintentional) changes to the text while reading, or due to the ways in which a reader’s cadence, accent, or tone of voice changes the perception of a poem. By annotating an audio poem with both a transcript of the audio and the original written poem, these changes can be visually tracked. Relevant fields include English, Literature, Poetry, and Writing.

**Ingredients**
- Audio recording of a poem being read aloud
- Transcript of the audio recording
- Original text of the poem
- Tags (optional)
- Analysis (optional)

**Method**
1. Listen to the audio recording of the poem and, without consulting the original text of the poem, write a transcript of the recording
  - Include line breaks, spacing, margins, punctuation, capitalization, italicization, or any other elements of a written poem according to how you interpret the audio recording of the poem
2. Use this transcript as well as the original text of the poem to create two sets of annotations for the audio recording, with **timestamps** for each annotation
3. Upload recording(s) to AVAnnotate and add **annotations**, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
4. Create and add **tags** to annotations, if desired, following Quick Start guide
5. If desired, create an additional page which includes a written analysis of the differences between the transcript and the original text

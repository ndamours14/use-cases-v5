**User Story:** As a conference participant, I want to present annotated recordings of conference events in one place so that I can document the conference for posterity.

**Description:** Events such as conferences, lecture series, and workshops are often recorded for posterity and future reference. An AVAnnotate project can serve as an archive of event recordings. Annotations can be added to recordings to provide additional context and insight, and tags can be used to make the event archive organized and searchable. Relevant to any fields hosting conferences, workshops, and other events.

**Ingredients**
- Multiple audiovisual recordings from an event
- Annotations
- Tags 

**Method**
1. Gather supplementary information, context, etc. for each audiovisual file, with associated **timestamps** for each piece of information
2. Upload audiovisual files to AVAnnotate and add information as **annotations**, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
3. Create **tags** that can be used to organize the collection, including things like subject, creator, date, etc.
4. Add **tags** to annotations, following Quick Start guide

**Example:** [SpokenWeb 2025: 360° Videos and Manifesto](https://ndamours14.github.io/360resounding/) by Natasha D’Amours

**User Story:** As a transgender community historian, I want to create an archive for a transgender community podcast so that I can share the podcast and supplemental information with a wider audience.

**Description:** A collection of audio or audiovisual content, such as the back catalogue of a podcast or web series, can be archived and shared as an AVAnnotate project. Annotations can be used to provide additional information about the audio files, and tags can be used to make the archive organized and searchable. Relevant fields include Media Studies, Cultural Studies, Podcast Production, and Video Production.

**Ingredients**
- A collection of audio or audiovisual files
- Tags
- Annotations (optional)

**Method**
1. Create **tags** that can be used to organize the collection, including things like subject, creator, date, etc.
2. If desired, gather additional information about each audio or audiovisual file, with associated **timestamps** for each piece of information
3. Upload audio or audiovisual files AVAnnotate and add **tags** to each file, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
4. Add supplemental information as **annotations** to each file, if desired, following Quick Start guide

**User Story:** As a researcher, I want to annotate video recordings with my observations so that I can gather organized data for my research.

**Description:** Observation as a research method can be done in-person, but can also be done via observation of video recordings. Whether the observation is structured (predetermined criteria) or unstructured (no predetermined criteria), observational notes can be included with the video as annotations. This can help with gathering, presenting, and/or coding the data. Relevant to any fields performing observational research.

**Ingredients**
- Video recording(s) to be observed for data
- Observations (structured or unstructured)
- Tags (optional)

**Method**
1. Watch video recording(s) and make note of all observations, either structured or unstructured, as well as associated **timestamps** of all observations
2. Upload video(s) to AVAnnotate and add observations as **annotations**, with associated **timestamps**, following [Quick Start: How to use AVAnnotate](https://av-annotate.org/how-to-use/) guide
3. Create and add **tags** to annotations, if desired, following Quick Start guide
  - Tags can be used to help with coding the observation data
